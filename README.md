# modulo_2
### En este Módulo 2, trabajamos los pilares fundamentales del análisis de datos y la optimización en R-Cran, enfocándonos en la transición de una programación básica hacia una eficiente y profesional. Los puntos clave que desarrollamos fueron:
## Fundamentos y Sintaxis:
Utilizamos comandos básicos, manipulamos variables, creamos vectores y exploramos datasets internos de R para comprender la estructura del lenguaje.
## Visualización Estadística:
Generamos gráficos esenciales como diagramas de dispersión, histogramas y curvas de densidad para interpretar distribuciones de datos.
## Implementación de Algoritmos:
Programamos manualmente soluciones clásicas como la Suma de Gauss, la serie de Fibonacci y el método de ordenamiento Burbuja, además de integrar bloques de código en Python para comparar entornos.
## Benchmarking y Performance: 
Este fue el eje central de nuestro trabajo. Utilizamos herramientas como microbenchmark y tictoc para medir y comparar la velocidad de nuestros algoritmos manuales frente a las funciones nativas de R, comprobando la superioridad de la vectorización.
## Aprendizaje No Supervisado: 
Implementamos el algoritmo K-means, analizando cómo varía su rendimiento según la cantidad de clusters definidos y entendiendo su lógica de agrupamiento por centroides.
## Creación de Presentaciones:
Exploramos las capacidades de R Markdown para generar diapositivas dinámicas y profesionales. Aprendimos a estructurar contenido visual, integrar código en tiempo real y exportar resultados a formatos de presentación, facilitando la comunicación efectiva de hallazgos técnicos.
## Formato Académico Profesional:
Integramos todo el contenido en un documento bajo la plantilla LNCS de Springer, dominando la configuración de archivos R Markdown, encabezados YAML y la gestión de márgenes para la generación de un PDF de calidad científica.
